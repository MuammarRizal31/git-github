fork atau forking adalah untuk membuat copy repo nya dari orang lain beserta historynya
    -jembatan antara repo original dan duplikatnya
    -memodifikasi terhadap repo original nya
    -bisa berkontribusi terhadap repo orang lain/memodifikasi codingan oranglain/membantu menyelesaikan masalah orang lain


yang membedakan fork dengan clone adalah    
    -jika clone untuk menduplikat langsung ke komputer nya
    -fork hanya menduplikat langsung ke githubnya saja/repository
    -fork dan clone tidak sama
    
dan yang membedakan download zip dengan fork adalah
    -download zip tidak ada sangkut pautnya dengan github
    -download zip hanya mendownload source codenya saja tidak kolaborasi
